Detailed Description

An interface class for storing states and looking them up in custom ways.

Note that only a way of adding and removing states is mandatory; there are no required query methods. This is because there could be many different kinds of query, depending on the approach taken for indexing.

You will need to define your own query methods if you implement this interface - your new Model class can then interface with those query methods as needed.
