const size_t abt::choosers::GpsChooserOptions::maxDimensions = 12
static

This is the maximum number of dimensions gps search expects at compile time.

Gps search uses template code which must be instanciated at compile time. This value determines up to which dimensionality the code is generated.

Setting this value high will slow down compilation and bloat the final executable.
