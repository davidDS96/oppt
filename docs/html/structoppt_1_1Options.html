Detailed Description

The base Options class for the ABT solver.

This is the data structure into which options are parsed; it is also used by the Model, Solver and Simulator to access the configuration settings, e.g. whether or not verbose output should be printed.

The values of numberOfStateVariables, minVal, and maxVal should be set by the model to correctly reflect the specific problem.

Additional configuration settings can be added by inheriting from this class.
