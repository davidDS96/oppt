Detailed Description

Represents a point within a discretized space.

This can be used to easily implement a simple, discrete state space by simply associating a unique ID with each individual state.

However, there need not be only one point in each bin; a bin can have as many or as few points as you prefer.

For convenience, the standard implementations for the equals(), hash() and print() are implemented simply by using the ID. If you want something more sophisticated, override them.
